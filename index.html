Prototips. Quin és l'objectiu?

L'objectiu d'aquest projecte és maquetar 10 prototips dissenyats a Figma,
utilitzant HTML i CSS per crear pàgines web responsives i atractives.
Cada prototip representa una interfície d'usuari diferent, amb l'objectiu de practicar i millorar les habilitats de maquetació web.
